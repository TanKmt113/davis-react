## DỰ ÁN CÁ NHÂN

### 1. Tích hợp hệ thống quản lý tồn kho từ Nhanh.vn sang Shopify

- Thiết kế pipeline tự động đồng bộ dữ liệu sản phẩm, giá, tồn kho từ **Nhanh.vn** → **Shopify**
- Xử lý thời gian thực: kiểm tra thay đổi, validate dữ liệu, cập nhật tự động
- **Kết quả:** Giảm **80–90%** thời gian cập nhật sản phẩm, hạn chế sai sót

### 2. Tự động hóa quy trình cập nhật sản phẩm với n8n

- Sử dụng **n8n** xây dựng workflow: ERP → xử lý logic → sync Shopify
- Thiết kế node xử lý: phân loại sản phẩm, mapping category, validate dữ liệu
- **Kết quả:** Rút ngắn thời gian cập nhật sản phẩm hàng loạt từ vài giờ xuống **10–15 phút**

### 3. Thiết kế và xây dựng website trên WordPress

- Tư vấn triển khai hoàn chỉnh website thương mại điện tử trên **WordPress + Woocommerce**
- Cấu hình hosting, theme, plugin, tối ưu tốc độ, bảo mật, SEO cơ bản
- **Kết quả:** Website ổn định, tốc độ tải cải thiện **30–50%**
